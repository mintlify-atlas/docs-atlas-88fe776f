# Welcome to Meridian

Meridian is a collaborative data analysis platform that makes working with data **tan-times faster** through real-time AI agents, streaming insights, and live-updating charts.

<Info>
  **Built for teams who need speed without sacrificing transparency.** Every analysis is tracked, every insight is reproducible, and every query happens in real-time.
</Info>

## What Makes Meridian Different

Traditional data tools force you to choose between power and simplicity. Julius AI and ChatGPT are easy to use but lack collaboration and reproducibility. Traditional BI tools are powerful but require SQL expertise and hours of setup.

Meridian bridges this gap:

## Key Value Propositions

### 1. Real-Time Collaboration

Powered by [Convex](https://convex.dev), Meridian delivers true real-time updates:
- Multiple users query simultaneously, all results sync instantly
- See query history and reasoning steps as they execute
- Live notifications when teammates discover insights or create charts

### 2. Transparent AI Reasoning

Unlike black-box AI tools, Meridian shows you **exactly how** it thinks:
- Agents break down requests into executable steps
- Watch reasoning stream live: "Reading column X..." → "Computing statistics..." → "Found pattern Y"
- Every generated query is visible and editable before execution

### 3. Server-Side Performance

Built with [TanStack Start](https://tanstack.com/start) and [DuckDB Node API](https://duckdb.org):
- DuckDB runs server-side (not browser WASM) for 10x faster queries
- Columnar storage optimized for analytical workloads
- Analyze millions of rows without browser memory limits

### 4. Zero Setup Required

<Steps>
  <Step title="Upload Data">
    Drag and drop CSV files or extract data from URLs using Firecrawl integration
  </Step>
  <Step title="Ask Questions">
    Type natural language queries or write SQL directly in the query editor
  </Step>
  <Step title="Get Insights">
    AI generates statistical findings automatically - outliers, trends, correlations
  </Step>
  <Step title="Share Results">
    Teammates see updates live. No exports, no emails, no waiting
  </Step>
</Steps>

## Technology Stack

Meridian combines best-in-class tools for maximum performance:

| Component | Technology | Purpose |
|-----------|-----------|----------|
| **Frontend** | [TanStack Start](https://tanstack.com/start) + [Mantine UI](https://mantine.dev) | Server-side rendering with React 19 and beautiful components |
| **Backend** | [Convex](https://convex.dev) | Real-time, type-safe serverless backend with subscriptions |
| **Analytics** | [DuckDB](https://duckdb.org) (Node API) | Vectorized OLAP engine for instant query execution |
| **Storage** | [Cloudflare R2](https://developers.cloudflare.com/r2/) | Cost-effective file storage for uploaded datasets |
| **AI Agents** | [Gemini 2.5 Flash](https://ai.google.dev) + Convex Agent Component | Streaming text/object generation with tool calling |
| **Data Extraction** | [Firecrawl](https://firecrawl.dev) | Turn any URL into structured CSV data |
| **Billing** | [Autumn](https://autumn.com) | Usage-based pricing with automatic metering |

<Note>
  The full stack is deployed on [Netlify](https://netlify.com) with Convex handling the backend. Authentication uses Convex Auth with support for Google OAuth.
</Note>

Meridian was built in public and shipped on **November 17, 2025**. The project combines:

- Real-time subscriptions (Convex presence API)
- AI agent orchestration (streaming with tool calls)
